# Carrières Dev — Press Kit
> Verified dev salaries in France, transparent and real
Official press kit for **[Carrières Dev](https://www.carrieres.dev)**. This repo is a permanent, linkable brand reference for journalists, directories, and integrations.
## About
Carrières Dev centralise plus de 1000 salaires de développeurs en France, tous vérifiés par email professionnel. Pour chaque entrée: salaire de base, bonus, equity, politique de télétravail, ancienneté, et région. Les développeurs peuvent comparer leur package au marché réel, préparer une négociation avec des chiffres solides, et accéder à du coaching carrière. Les recruteurs y trouvent une grille de référence fiable pour construire leurs offres.

## Long-form description
Le marché du salaire développeur en France souffre d'un problème d'opacité: tout le monde devine, personne ne sait. Carrières Dev corrige ce problème avec 1000+ entrées vérifiées par email professionnel, couvrant base, bonus, equity, et télétravail. Les développeurs préparent leurs négociations avec des chiffres réels plutôt qu'avec des rumeurs Slack, et les recruteurs construisent leurs grilles à partir de données vérifiées plutôt que des estimations Syntec datées.
